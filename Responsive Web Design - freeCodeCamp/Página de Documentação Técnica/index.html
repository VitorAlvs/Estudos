O que é HTML?

Primeiramente, HTML não é uma linguagem de programação, é uma linguagem de marcação e é utilizada para definir a estrutura de seu conteúdo.

Consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira.

As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante.

Principais partes

  1. Tag de Abertura: O elemento em questão é identificado pelo seu nome entre parênteses angulares de abertura e fechamento. Essa notação indica o ponto de partida do elemento ou o início de sua aplicação, neste contexto específico, indicando onde o parágrafo começa;
  2. Tag de Fechamento: Isso é equivalente à tag de abertura, com a única diferença de que possui uma barra antes do nome do elemento. Essa notação indica o ponto de término do elemento, no caso, o fim do parágrafo. Omissões da tag de fechamento são erros frequentes cometidos por iniciantes e podem resultar em comportamentos inesperados;
  3. Conteúdo: O conteúdo do elemento;
  4. Elemento: Tag de abertura, de fechamento, e o conteúdo formam o elemento;
  5. Atributos: Inclui informações adicionais sobre o elemento que não devem ser exibidas no conteúdo final.
<p id="gato">Meu gato é preto</p>

No exemplo acima, < é uma tag de abertura ; > é uma tag de fechamento ; o conteúdo dentro de <p> e </p> são o conteúdo ; todo o texto forma o elemento e id="gato" é um atributo;

Aninhar elementos

Você pode colocar elementos dentro de outros elementos também — isso é chamado de aninhamento. Se quiséssemos afirmar que nosso gato é muito mal-humorado, poderíamos envolver a palavra "muito" em um elemento <strong>, o que significa que a palavra deve ser fortemente enfatizada:

<p>Meu gatinho é <strong>muito</strong> mal humorado.</p>

Entretanto, é crucial assegurar que seus elementos estejam devidamente organizados em camadas. No caso do exemplo mencionado, começamos com a abertura do elemento <p> seguido pelo elemento <strong>. Portanto, a sequência de fechamento deve seguir a mesma ordem: primeiro o elemento <strong> e, em seguida, o elemento <p>.

Elementos vazios

Alguns elementos não possuem conteúdo e são chamados de elementos vazios. Considere o elemento que temos na nossa página HTML:

<img src="imagens/firefox-icon.png" alt="Minha imagem de teste" />

Ele contém dois atributos, mas não há tag de fechamento, e não há conteúdo interno. Isso acontece porque um elemento de imagem não envolve conteúdo para ter efeito em si mesmo. Sua proposta é incorporar uma imagem na página HTML no lugar que o código aparece.

Imagens
<img src="imagens/firefox-icon.png" alt="Minha imagem de teste" />

Como dissemos antes, o elemento <img/> incorpora uma imagem na nossa página na posição que aparece. Isso é feito pelo atributo src (source), que contém o caminho para nosso arquivo de imagem.

Incluímos também um atributo alt (alternative). Neste atributo, você especifica um texto descritivo para usuários que não podem ver a imagem.
